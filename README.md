# Финансовый аналитик: анализ транзакций и отчёты

Проект реализует набор сервисов и отчётов для анализа банковских транзакций. Позволяет получать аналитику по расходам, рассчитывать кешбэк, работать с «Инвесткопилкой» и формировать различные отчёты.

## Функциональность

Проект включает следующие модули:

**Веб‑страницы:**
* «Главная» — показывает приветствие, данные по картам, топ‑5 транзакций, курсы валют и акций.
* «События» — предоставляет детализацию расходов и поступлений за выбранный период.

**Сервисы:**
* «Выгодные категории повышенного кешбэка» — анализирует, какие категории принесут больше кешбэка.
* «Инвесткопилка» — рассчитывает сумму накоплений через округление трат.
* «Простой поиск» — ищет транзакции по запросу в описании или категории.
* «Поиск по телефонным номерам» — находит транзакции с номерами телефонов в описании.
* «Поиск переводов физическим лицам» — выделяет переводы с указанием ФИО в описании.

**Отчёты:**
* «Траты по категории» — показывает траты по выбранной категории за последние 3 месяца.
* «Траты по дням недели» — средние траты в каждый день недели за 3 месяца.
* «Траты в рабочий/выходной день» — средние траты в рабочие и выходные дни за 3 месяца.
